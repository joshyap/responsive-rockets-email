This is purely a demo responsive email. It is made with the mjml framework.

MJML stands for mailjet markup language and is an open source framework for making responsive emails that are email client agnostic.

This demo email is based on my favorite basketball team, the Houston Rockets.

Truly repsonsive
Frontend development has progressed so far, but making responsive emails can feel like the dark ages. MJML makes it easy to create email templates that will work in basically any email client and view size.
Prove it

I generated a static version of this email and hosted it with surge.sh for easy viewing. Additionally, I used test.at (a free alternative to "Email on Acid" and "Litmus") for testing to see how things would look for recipients of my email template.

It actually looked perfect on the first try without any real issues that needed to be fixed.
